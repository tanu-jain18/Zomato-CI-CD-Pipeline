# 🍽️ Zomato Clone DevSecOps CI/CD Pipeline

## 📌 Project Overview

This project demonstrates the implementation of a **DevSecOps CI/CD Pipeline** for a Zomato Clone web application using modern DevOps and security tools. The project focuses on automating code integration, security analysis, and deployment processes.

The application is deployed on AWS EC2 and integrated with Jenkins, SonarQube, Docker, and OWASP Dependency Check to simulate a real-world DevSecOps workflow.

# ⚙️ Jenkins Pipeline Stages

## ✅ Checkout SCM

Source code is fetched from GitHub repository.

## ✅ Tool Installation

Required tools like NodeJS are installed automatically in Jenkins.

## ✅ Install Dependencies

Project dependencies are installed using npm.

## ✅ SonarQube Scan

Static code analysis is performed to detect bugs, vulnerabilities, and code smells.

## ✅ OWASP Dependency Check

Checks third-party dependencies for known vulnerabilities.

## ✅ Docker Stage

Docker integration stage added for containerization workflow.

## ✅ Deployment

Application successfully deployed and executed on AWS EC2 instance.
